# 4tential 

> **운동 초보자를 위한 미래 체형 예측 기반 동기부여 플랫폼**

운동을 시작하고 싶지만 결과가 보이지 않아 쉽게 포기하시나요?  
저희의 프로젝트는 당신의 **미래 체형**을 미리 보여주고, 지속적인 운동 동기를 제공하는 플랫폼입니다.

## 🎯 타겟 사용자
- 운동을 시작하고 싶지만 막연한 초보자
- 빠른 결과를 보지 못해 운동을 포기했던 사람
- 구체적인 목표 설정이 어려운 사람
- 시각적 동기부여가 필요한 사람

## 💡 주요 기능
### 1. 메인 및 프로필 설정 (Main & Profile)
- **동기부여 명언 제공**: 매일 접속 시 운동 욕구를 자극하는 새로운 명언을 메인 화면에 노출
- **맞춤 프로필 설정**: 이름, 키, 몸무게, 근육량, 체지방률 상세 신체 정보 입력
- **운동 목표 선택**: 4가지 목적(다이어트 / 체력향상 / 근력향상 / 체형교정) 중 개인별 목표 설정

### 2. 미래 체형 시뮬레이션 (My Body)
- **체형 변화 예측**: 입력된 신체 정보와 운동 목표를 알고리즘으로 분석하여 미래 모습 시각화
- **단계별 변화 확인**: 현재 상태와 3개월, 6개월, 1년 후의 변화된 체형을 직관적으로 비교
- **동기부여 강화**: 변화될 자신의 모습을 미리 눈으로 확인하여 지속적인 운동 의지 고취

### 3. 맞춤형 운동 추천 및 식단 추천(Workout Recommendation)
- **목적별 루틴 제공**: 사용자가 설정한 목표에 최적화된 운동 리스트 자동 추천
- **상세 가이드 안내**: 각 운동별 권장 시간, 세트 수, 반복 횟수, 휴식 시간 정보 제공
- **영상 학습 지원**: 유튜브 검색 연동을 통해 정확한 운동 자세와 동작 학습 가능

### 4. 식단 정보 검색 (Food Search)
- **영양 검색**: 식품 영양 API를 연동하여 실시간 영양 정보 조회하여 원하는 식단 메뉴 검색
- **상세 성분 정보**: 단순 칼로리뿐만 아니라 탄수화물, 단백질, 지방 등 핵심 영양소 데이터 제공

## 📦 설치 및 실행
본 프로젝트는 NPM과 Yarn을 모두 지원합니다.
정상적인 식단 검색 기능 사용을 위해, 프로젝트 실행 전 반드시 API 서버를 먼저 실행해주세요.
 💡참고사항: API 키가 포함된 .env 파일은 프로젝트 server 폴더 내에 미리 설정되어 있으므로, 별도로 생성하실 필요가 없습니다.

### 1. 의존성 패키지 설치 (Dependencies)
프로젝트 루트 경로에서 다음 명령어를 입력하여 필요한 패키지를 설치합니다.
```bash
# npm 사용 시
npm install

# yarn 사용 시
yarn install
```

### 2. 서버(Server) 실행 (터미널 1)
```bash
# 1. 서버 디렉토리로 이동
cd server

# 2. 서버 실행 (.env 설정 자동 적용됨)
node server.mjs
```
(서버가 실행된 상태에서 터미널을 끄지 말고 유지해주세요.)

### 3. 클라이언트(Client) 실행 (터미널 2)
새로운 터미널을 열고 프로젝트 루트 경로에서 실행합니다.
```bash
# npm 사용 시
npm start

# yarn 사용 시
yarn start
```
(잠시 후 브라우저가 자동으로 실행됩니다.)
