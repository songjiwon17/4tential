# 4tential 

> **운동 초보자를 위한 미래 체형 예측 기반 동기부여 플랫폼**

운동을 시작하고 싶지만 결과가 보이지 않아 쉽게 포기하시나요?  
저희의 프로젝트는 당신의 **미래 체형**을 미리 보여주고, 지속적인 운동 동기를 제공하는 플랫폼입니다.

## 📌 팀 소개

4tential(포텐셜)은 'Four + Potential'의 합성어로, 
4명의 잠재력이 모여 무한한 가능성을 만들자! 라는 취지에서 결정된 팀 이름입니다.

## 주요 기능

### 1. 미래 체형 시뮬레이션 (나의 체형 페이지)

**현재 신체 정보 입력**

사용자 프로필 관리에서 입력한 정보를 기반으로 <br>
현재 체형 상태 분석


**시각적 비교**

시간대별로 변화하는 체형 이미지를 통해 변화 정도를 직관적으로 확인


### 2. 맞춤형 운동 추천 (운동 추천 페이지)

**목표 기반 운동 프로그램**

설정한 운동 목표(다이어트 / 체력향상 / 근력향상 / 체형교정) 에 따른 최적화된 운동 추천


**운동 가이드 연동**

YouTube 영상을 통한 상세한 운동 방법 안내
정확한 자세와 동작을 영상으로 학습 가능


**운동 정보 제공**

각 운동의 이름, 권장되는 운동 목적, 운동 시간 및 반복 횟수 가이드



### 3. 식단 검색 (식단 검색 페이지)

**음식 영양정보 검색**

실시간 음식 데이터베이스 검색 </br>
음식명으로 간편하게 영양 정보 조회


**상세 영양 성분 표시**

칼로리, 탄수화물, 단백질, 지방 등 주요 영양소 데이터 제공


### 4. 사용자 프로필 관리

**개인 정보 관리**

이름, 키, 몸무게, 근육량, 체지방률의 사용자 데이터 입력

**목표 설정**

다이어트 / 체력향상 / 근력향상 / 체형교정 중 운동 목표 선택

## 🎯 타겟 사용자

## 📦 설치 및 실행

**실행 전, 식단 검색 기능을 위해 서버의 실행이 필요합니다!**

```bash

npm install

node server.mjs

# 3. 실행
또다른 터미널 실행 후,
npm start
```
